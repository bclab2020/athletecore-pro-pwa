# AthleteCore Pro × Dior Sport - ラグジュアリーAI姿勢分析システム

## 🚀 プロジェクト概要

**AthleteCore Pro × Dior Sport Edition** は、最先端のAI技術と**完璧なDior Sport統一デザイン**を融合させたプログレッシュWebアプリケーション（PWA）です。MediaPipe技術を活用したリアルタイム姿勢分析と、ラグジュアリーブランド級の洗練されたデザインシステムにより、プレミアムアスリート体験を提供します。

### ✨ 主要な特徴

- **🎯 リアルタイムAI姿勢分析**: MediaPipeを活用した高精度な姿勢検出
- **🌟 プレミアムダイナミックデザイン**: パーティクルアニメーション、3D変形エフェクト、リッチグラデーション
- **📱 完全モバイル最適化**: モバイルファーストアプローチによる100%モバイル互換性  
- **💫 インタラクティブ体験**: タッチ対応パーティクル生成、流体的アニメーション
- **🛡️ パフォーマンス最適化**: デバイス別自動調整、省電力モード対応
- **💾 データ管理**: 分析結果の自動保存・履歴管理・統計表示
- **📲 PWA対応**: オフライン機能、アプリライクな体験、ホーム画面追加可能

## 📊 分析指標詳細

### 姿勢分析アルゴリズム
```javascript
// 主要分析指標
- shoulderLevel: 肩の水平レベル偏差
- hipLevel: 腰の水平レベル偏差  
- headAlignment: 頭部と肩の中心軸からの偏差
- spineAlignment: 脊椎の直線性評価
```

### スコア計算方式
- **姿勢スコア**: (100 - 肩傾き×2 - 頭部偏差×3)
- **バランススコア**: (100 - 腰傾き×3 - 肩傾き)
- **アライメントスコア**: (100 - 頭部偏差×5)
- **安定性スコア**: 動的安定性評価（90-100）
